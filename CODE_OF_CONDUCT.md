# Code of Conduct

## Our Commitment

We are committed to providing a welcoming and inclusive environment for contributors, users, and maintainers. This project follows the Contributor Covenant Code of Conduct.

## Expected Behavior

- Be respectful and constructive in all interactions
- Accept constructive criticism professionally
- Focus on what is best for the community and the project
- Show empathy toward other community members

## Unacceptable Behavior

- Harassment, discrimination, or discriminatory language
- Personal attacks, trolling, or public insults
- Publishing others' private information without permission
- Any conduct that would be inappropriate in a professional setting

## Reporting and Enforcement

If you encounter behavior that violates this code of conduct:
1. Reach out to the maintainers privately if a contact channel is available
2. Or file a private security advisory through GitHub

All complaints will be reviewed and investigated fairly. Maintainers who do not follow the code of conduct may face temporary or permanent restrictions.

## Scope

This code of conduct applies to:
- Project repositories and issue trackers
- Community events and discussions
- Online and offline interactions related to the project
